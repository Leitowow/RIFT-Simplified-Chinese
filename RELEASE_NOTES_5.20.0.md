# RIFT 中文版 v5.20.0

发布日期：2026-05-12

## ✨ 新增功能

- 新增 **制造助手** 窗口（Neocom 入口）
- 支持蓝图按 **英文名 / 中文名 / Type ID** 查询
- 支持自动补全与候选确认，输入更快
- 支持配置数量、附加成本、ME、税率、结构与改装件参数
- 支持结构化结果展示与一键复制

## 📈 计算能力增强

- 对接 EveCookbook 建造成本接口，输出标准化成本字段
- 结合产出映射与市场估价，补充买/卖两档预估收益
- 增加利润与利润率展示，方便快速判断是否开工

## 🌐 本地化与可用性优化

- 补充并接入蓝图中文对照数据
- 优化中英文混合输入时的匹配与回退

## 🔔 版本更新提醒恢复

- 启动时自动检查 GitHub Releases 最新版本
- 检测到新版本后即时提示“版本过旧”（仍为手动更新）

## ✅ 兼容性

- 本次为向后兼容的小版本升级：`5.19.x` -> `5.20.0`
- 不影响现有配置结构
- 若网络不可用，在线更新检测会静默失败，不影响主功能使用
